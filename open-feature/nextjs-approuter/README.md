## Flagsmith with Next.js/AppRouter and OpenFeature

The repository for the [live stream tutorial](https://www.youtube.com/watch?v=u9TjbtZX4Zg) with [@eddiejaoude](https://twitter.com/eddiejaoude).

## Getting started


**Install**

```
npm i
```

**Run**
```
npm run dev
```
